React Hooks中使用debounce

2022-05-27 1 min read # 前端小技巧 # react

使用场景

  • 依赖频繁触发，但只想执行一次

方式

useLayoutEffect(()=>{
    const fn = () => {
        console.log('do something')
    }
    const debounceFn = debounce(fn, 100)
    debounceFn()
    return () => {
        debounceFn.cancel()
    }
},[dep1, dep2, dep3])

注意事项

需要在卸载的时候进行cancel
